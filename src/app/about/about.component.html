ABOUT THIS PROJECT

This application is designed for practice purpose only.

Technologies used:

  • HTML5, CSS3, Typescript & Jquery
  • Angular 11
  • Bootstrap 4
  • GITHUB for code base
  • NETLIFY for deployment

API's used:

  • OMDB API - for fetching imdb rating, movie poster specifically
  • TMDB API - for fetching Movie details, Cast details, youtube video ids, more photos.
  • Youtube API - for embedding youtube videos related to the movie searched.
